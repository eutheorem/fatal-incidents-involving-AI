# Summary

This repository is for storing and distributing supplementary material of the article "Fatal incidents involving generative AI" published at https://eutheorem.com/fatal-incidents-involving-generative-ai/. The repository contains R code, csv files used in the analysis, and figures. 

# Folders and files

## csv/

- **concepts_modified.csv**: The file includes numerical values for vertical justification of text in the graphical abstract. 

## figures/

The folder contains figures and graphical abstract.

- **AI_death_GA.png**: Graphical abstract of the article. 

- **fig_AI_death_concept.png**: Figure 1.

## OECD/xlsx

The folder contains data files of fatal incidents involving AI, downloaded from OECD AIM as xlsx files. Each of the three files contains 100 event records, sorted differently.
